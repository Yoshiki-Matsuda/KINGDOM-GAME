## 1. ゲーム全体構造

Kingdom Conquest は **MMO リアルタイムストラテジー + カードコレクション** のモバイルゲーム。SEGA / ヘッドロック開発。iOS 2011年1月、Android 2011年12月リリース。2014年7月31日にサービス終了。

プレイヤーは魔獣の大陸「マグナ」に降り立った開拓者となり、拠点の施設を建設・強化しつつ、入手した魔獣カードを用いて周囲の領地を制圧していく。最終目標はマップ上の **デブリズタワー** 全てを同盟として占拠すること。

### 基本ループ

```
魔獣カード入手 → カード育成・合成 → ユニット編成 → 領地攻撃 → 領地拡大 → 資源増加 → 施設強化 → (ループ)
```

### 2パート構成（参考）

| パート | 内容 | 本プロジェクト |
|---|---|---|
| **SLG パート** | 内政・領地攻撃・同盟戦 | **採用** |
| ACT パート | ダンジョン探索 (ARPG) | 不採用 |

## 2. マップ・領地システム

### マップ構造

- 全プレイヤーが共有する大規模マップ（1ワールド = 1「シャード」）
- 各マスには **レベル** (1〜9) が設定される

### 領地の色分け（出典: [土地・拠点](https://seesaawiki.jp/kingdomconquest/d/%c5%da%c3%cf%a1%a6%b5%f2%c5%c0)）

| 枠無し | 空き地（NPC が守備） |
| 青 | 自領土 |
| 緑 | 自同盟員の領土 |
| 橙 | 配下同盟の領土 |
| 紫 | 親同盟の領土 |
| 赤 | 他同盟の領土 |

### 領地の資源産出

出典: [建物一覧](https://seesaawiki.jp/kingdomconquest/d/%b7%fa%ca%aa%b0%ec%cd%f7)

- 領地には **森林・岩山・鉄鉱山・穀物地帯** の資源タイルが含まれる
- 拠点化した場合: 資源タイル1つにつき **30/時間** を産出
- 拠点化せず占有のみ: 資源タイル1つにつき **5/時間** を産出
- 高レベル領地ほど資源タイル数が多い（例: Lv5木材エリアの場合、木材28マス×5 = +140/時間）

### 攻撃可能条件

- **自分の拠点（本拠地 or 前線基地）に隣接した領地** のみ攻撃可能
- 同盟員・配下同盟の領地も隣接として扱える（経由可能）
- 親同盟の領地は隣接判定に含まれない
- 他プレイヤーの領地・拠点も攻撃可能（PvP）。耐久値を0にすると占領
- 同盟員の領地には「攻撃」or「援軍」を選択可能。攻撃した場合は NPC が出現し、倒すと耐久値が回復

## 3. 拠点・施設システム

### 拠点の種類

| 種類 | 中央建物 | 最大Lv | 特徴 |
|---|---|---|---|
| **本拠地** | 城 | Lv20 | ゲーム開始時に自動配置。施設スロット多い |
| **前線基地** | 砦 | Lv15 | 占領した領地に建設。本拠地より施設スロット少ない |

### 拠点建設の条件（出典: [土地・拠点](https://seesaawiki.jp/kingdomconquest/d/%c5%da%c3%cf%a1%a6%b5%f2%c5%c0)）

拠点の建設には **統治力** が必要。統治力はプレイヤーレベルで決まる。

| 建設数 | 必要統治力 |
|---|---|
| 1 | 17 |
| 2 | 35 |
| 3 | 54 |
| 4 | 80 |
| 5 | 112 |
| 6 | 150 |
| 7 | 195 |
| 8 | 248 |
| 9 (MAX) | 310 |

推奨: 1つ目は Lv3〜4 の領地に建設。生産拠点としては Lv7 が理想。

### 資源（4種）

| 資源 | 生産施設 | 用途 |
|---|---|---|
| **糧（食料）** | 畑 | 魔獣の生産・維持、施設建設 |
| **木** | 木材工房 | 施設建設全般（序盤の主要消費） |
| **石** | 石切り場 | 施設建設（中盤以降で要求増大） |
| **鉄** | 製鉄所 | 施設建設（上位施設で必須） |

- 各生産施設は **隣接する資源タイル** の産出量を増加させる（畑のみ単独で建設可能）
- 資源は **1時間ごと** に生産施設のレベルに応じた量が加算される
- 資源には **貯蔵上限** があり、資源貯蔵庫で上限を増やす

- 1つの施設の建設中は他の施設を建設・強化できない（建設キューは1）

- 施設の破壊は1時間かかり、投入した資源は戻らない

## 4. 魔獣カードシステム

### カードの入手方法

1. **ダンジョンクリア報酬**（ACTパート） → 本プロジェクトでは領地攻撃時のドロップに置換
2. **探索**（→ セクション11）
3. **オークション**（プレイヤー間取引）
4. **課金パック**（ダンジョンパック、クリスタルで購入）

### カードの基本ステータス

| ステータス | 説明 |
|---|---|
| **種族** | 獣族・亜人族・不死族・精霊族・巨人族・魔族・龍族・天使族の **8種** |
| **レアリティ** | C（コモン）, UC（アンコモン）, R（レア）, SR（スーパーレア） |
| **コスト** | 1.5 / 2.0 / 2.5 / 3.0 / 3.5。ユニット編成時のコスト上限を消費 |
| **攻撃力** | 通常攻撃・攻撃スキルのダメージ基準値 |
| **防御力** | 被ダメージ軽減の基準値 |
| **知力** | スキル効果の強さに影響。スタートアップ・トリック系で特に重要 |
| **素早さ** | 行動順の決定。移動速度にも影響 |
| **射程** | 攻撃可能な距離（1=近接, 2=中距離, 3=遠距離） |
| **魔獣数** | HP に相当。ダメージで減少し、0 で戦闘不能。最大 9,999 |
| **占拠力** | 勝利時に敵拠点の耐久値を削る量 |
| **スタミナ** | 出撃に必要。時間で回復。MAX 出撃で経験値ボーナス |

### 種族間の相性（出典: [戦闘(Ver.C)](https://seesaawiki.jp/kingdomconquest/d/%c0%ef%c6%ae%28Ver%2eC%29)）

巨人 = 魔 = 龍 > 亜人 = 不死 = 精霊 > 獣 という大まかな力関係がある（ダメージ補正）。要検証とされていた。

### カードの成長

- 魔獣カードには **レベル** があり、戦闘で経験値を獲得して成長
- レベルアップ時に **ステータスポイント** を振り分けてカスタマイズ
- カードごとに成長タイプ（成長単位）が異なる
- R, SR のカードは初期スキルレベルが **3** からスタート
- **覚醒**（iOS 4th Season以降）: 3スキル全てが Lv5 以上の状態で合成すると、一定確率で覚醒。Lv99 を超えてレベルアップ可能に。全スキル Lv10 なら覚醒率 100%

## 5. ユニット編成（出典: [ユニット編成](https://seesaawiki.jp/kingdomconquest/d/%a5%e6%a5%cb%a5%c3%a5%c8%ca%d4%c0%ae)）

### 編成構造

1ユニットは **最大3種の魔獣** で構成される。3つの配置スロットがある。

```
敵ユニット                       自ユニット
┌────────┐                      ┌────────┐
│ Front  │ ← 射程1で到達 →      │ Front  │
│ Back   │ ← 射程2で到達 →      │ Back   │
│ Leader │ ← 射程3で到達 →      │ Leader │
└────────┘                      └────────┘
```

### 配置と射程の関係

| 配置 | 役割 | 攻撃に必要な射程 |
|---|---|---|
| **Front (F)** | 前衛 | 射程1以上 |
| **Back (B)** | 中衛 | 射程2以上 |
| **Leader (L)** | 指揮官（撃破で敗北） | 射程3以上 |

- 射程が足りないと「射程が足りないので攻撃出来ない」と表示され、行動できない
- Front がいる状態で相手の Leader を攻撃するには射程2以上が必要
- パッシブスキル「バーストレンジ」「テイルウィンド」で射程を延長可能

### ユニットコスト制

初期コスト上限は **4.0**。

| コスト上限 | 実現方法 | 現実的な編成例 |
|---|---|---|
| 4.0 | 初期値 | [2.5のみ単騎] or [2.5 + 1.5] |
| 4.5 | 課金+0.5 (7日/50CP) | [2.5 + 2.0] |
| 6.5 | 課金+0.5 × 5効果 | [2.5 + 2.5 + 1.5] 3体ユニット |
| 7.0 | 伝承資料庫Lv5 (+3.0) | 3体ユニットを十分運用可能 |
| 9.5 | 通常最大 | コスト上限まで埋めること自体が難しい |
| 10.5〜11.5 | Season進行で上限増加 | 4th Season以降 |

### ステータスボーナス（出典: [ユニット編成](https://seesaawiki.jp/kingdomconquest/d/%a5%e6%a5%cb%a5%c3%a5%c8%ca%d4%c0%ae)）

| ボーナス | 効果 |
|---|---|
| **種族ボーナス** | 同ユニット内に同種族2体で+5%、3体で+10% |
| **研究所Lvボーナス** | 研究所のLv数値がそのまま%ボーナス（最大+10%） |
| **戦神の祠/守護神の祠** | 拠点内の魔獣に攻防+2%/Lv。Lv5以降で全拠点に+効果。拠点内最大+20%、全体最大+3% |
| **デブリズBonus** | デブリズタワー占拠で対応種族の全ステータス+20% |
| **称号** | 特定の種族編成で追加ボーナス（巨人族・魔族・龍族で実装） |

最大ステータス補正: 攻撃力・防御力 = +70%、知力・素早さ = +40%、占拠力 = +20%

### ユニットの解散

- ユニットから外された魔獣は、スタミナが300以上なら300に、魔獣数が100以上なら100に減らされる
- 拠点間でユニットを組み直す際は注意が必要

## 6. SLG 戦闘システム（Ver.C 準拠、出典: [戦闘(Ver.C)](https://seesaawiki.jp/kingdomconquest/d/%c0%ef%c6%ae%28Ver%2eC%29)）

### 勝利条件

| 側 | 勝利条件 |
|---|---|
| **攻撃側** | 防御側の **Leader** を戦闘不能にする |
| **防御側** | 攻撃側の Leader を戦闘不能にする、**または 8ターン経過** |

スキル「エスケープ」によりターン数が変動する場合がある。

### 戦闘の流れ

#### 1. スタートアップフェーズ (Start Up)

- **素早さが低い順** に行動
- スタートアップスキルを持つ魔獣がスキルを発動
- スキルの効果対象は一体ずつ確率抽選（スキルレベルで確率が変動）
- ステータスに関わるスタートアップスキルの効果は **知力・コスト・スキルレベル** で上昇
- 「クイックシフト」「ラピッドトリック」による素早さ増減はスタートアップ中の行動順には影響しない

#### 2. 通常ターン (Turn 1〜8)

- **素早さが高い順** に行動（スタートアップとは逆順）
- 素早さが同じ場合は **ランダム**
- 全ての生存魔獣は毎ターン必ず行動順が回ってくる

各魔獣の行動:
1. **アクティブスキル判定** — Skill1 → Skill2 → Skill3 の順に発動率で抽選。先に発動したら後は判定しない
2. スキルが発動した場合 → **そのターンの通常攻撃は行わない**
3. スキルが発動しなかった場合 → **通常攻撃** を実行
4. 射程内に攻撃可能な敵がいない場合は行動なし（「射程が足りない」）

> 重要: アクティブスキルが発動すると、そのターンの攻撃権を失う。運が悪いとスキルが発動しすぎてターン数が経過し、防御側有利になることがある。

### ダメージ計算

#### 通常攻撃

- 基本ダメージは **攻撃力 vs 防御力** で決定
- **魔獣数** がダメージの大きさに影響（多いほど強い）
- 射程内に攻撃可能な敵が複数いた場合、攻撃対象は **ランダム** で選ばれる
- 種族間の相性による補正あり

ダメージの上限・下限:
- 攻撃力 > 防御力 でも、最大ダメージは **魔獣数の約 1.1 倍** まで
- 防御力 > 攻撃力 でも、**最低ダメージ** が設定されている:
  - 魔獣数 ~1000: 最低ダメージ ≈ 魔獣数の 10%
  - 魔獣数 9999: 最低ダメージ ≈ 326（約3%）
  - 魔獣数 1: ダメージ **5固定**

#### 攻撃スキル

- 通常攻撃ダメージに補正を掛けた値がスキルダメージになる（詳しい計算式は未判明とされている）
- スキルレベル・知力で倍率が変動
- スキルによって対象が「単体」「Front全体」「敵全体」等に分かれる
- スキル効果範囲には「スキル固有のもの」と「魔獣の射程に依存するもの」がある

### 連続戦闘（出典: [戦闘(Ver.C)](https://seesaawiki.jp/kingdomconquest/d/%c0%ef%c6%ae%28Ver%2eC%29)）

塔・拠点・高レベルの土地では、敵ユニットが複数出てくる。

- 各ウェーブごとに 8 ターン制限が独立して適用
- **減った魔獣数はそのまま引き継ぎ**
- **スキルの効果はウェーブ間でリセット** される
- 戦闘不能になった魔獣は **以降のウェーブに参加できず、経験値も入らない**

### 戦闘終了後の処理

- 相手に与えたダメージに応じて **経験値** が付与される
- 経験値はダメージ量・Lv・種族・レアリティで変動
- 回復スキルで回復した分のダメージは経験値に含まれない
- スタミナが多いほど経験値にボーナス。スタミナ200以下だと経験値0
- 攻撃側勝利時、味方の **占拠力の合計** が敵の耐久値から差し引かれる（戦闘不能の魔獣の占拠力も含む）

## 7. スキルシステム（出典: [スキル](https://seesaawiki.jp/kingdomconquest/d/%a5%b9%a5%ad%a5%eb)）

### スキルスロット構成

各魔獣は最大 **3つ** のスキルスロットを持つ。

| スロット | 種別 | 変更 |
|---|---|---|
| **Skill 1** | 初期スキル（魔獣固有） | 変更不可 |
| **Skill 2** | 合成で追加 | スキル合成で付与 |
| **Skill 3** | 合成で追加 | スキル合成で付与 |

### スキルの3分類

#### スタートアップスキル

- **発動タイミング**: 戦闘開始時（Start Up フェーズ）
- **発動条件**: 所持していれば **必ず発動**（効果判定は一体ずつ確率抽選）
- **効果対象**: 敵味方どちらか全体
- **効果例**: ステータス強化、ステータス弱化、素早さ増減
- **効果依存**: 知力・コスト・スキルレベル

#### アクティブスキル

- **発動タイミング**: 通常ターンの行動時
- **発動条件**: スキルごとに設定された **発動率** で確率判定
- **発動時**: そのターンの **通常攻撃を行わない**
- **複数アクティブスキル**: Skill1 → Skill2 → Skill3 の順に抽選。先に発動したら以降は判定しない。3つ全てアクティブだと Skill3 の発動確率は大幅に下がる
- **効果例**: 単体攻撃、範囲攻撃、回復、バフ、デバフ、状態異常
- **効果範囲**: スキル固有のものと、魔獣の射程に依存するものがある

#### パッシブスキル

- **発動タイミング**: 常時
- **発動条件**: 常時発動（確率判定なし）
- **効果例**: 攻撃力 UP、防御力 UP、射程延長（バーストレンジ、テイルウィンド）、状態異常耐性
- 同系統のパッシブ効果は **加算** される（例: 攻撃オーラ + 攻撃スピリット）
- **スピリット系** はオーラ系より効果が高い

### スキルレベル

- スキルにはレベル (1〜10) があり、**カード合成** で上げられる
- レベルが上がると:
  - **発動率が上昇**（アクティブスキル）
  - **効果値が増大**（全スキル共通）
  - スタートアップスキルの **効果発生確率が上昇**

### バフ・デバフのルール（出典: [戦闘(Ver.C)](https://seesaawiki.jp/kingdomconquest/d/%c0%ef%c6%ae%28Ver%2eC%29)）

- **重ね掛け**: 同じ効果を掛け直すと **前の効果が消えて新しい効果に上書き**
- **弱い効果で上書き**: 既に同じ効果が全対象に掛かっている場合、弱い効果での上書きは **発動自体が抑制** される
- **バフ + デバフ**: 同じステータスに強化と弱化が掛かると、効果の強さに関係なく **両方が無効化**
- **知力バフ**: 知力バフスキル使用時、パッシブスキル（オーラ等）の効果は無視される。また魔獣数100未満だと大幅弱体化

### 主な状態異常

| 状態異常 | 効果 | 代表スキル |
|---|---|---|
| **毒** | 毎ターンダメージ。ガード・バリア無視。魔獣数差で効果変動。効果ターンやダメージ量はスキルで異なる | ポイズンフィールド、ヴェノムヘイズ |
| **混乱** | 味方を攻撃する可能性。自分は攻撃しない | テラーフィールド、カタクリズム |
| **魅了** | 100%相手側に有利な行動。自分自身も攻撃対象 | テンプテーション |
| **沈黙** | アクティブ・スタートアップスキルが発動不可 | サイレンスフィールド |
| **スタン** | 1ターン行動不能。ガード・待機状態を解除 | ヒステリックホールド、ダイアモンドダスト |
| **挑発** | 掛けた魔獣が射程内にいれば強制攻撃。射程外だと効果が出にくい | プロヴォーク |

- 全ての状態異常は **回復スキル**（ラピッドヒール等）や **デバフ除去スキル**（インビンシブル、メイルシュトローム等）で解除可能
- 毒・混乱・沈黙・スタンには専用の **無効化・反射スキル**（○○リフレクト等）が存在

### 特殊状態（バフ・デバフいずれにも属さず、除去スキルの影響を受けない）

| 状態 | 効果 | 代表スキル |
|---|---|---|
| **増殖** | 一時的に魔獣数を水増し。戦闘終了後に元に戻る。増加分がダメージの盾になる | ダイナストシフト、ラストモーメント |
| **リザレクション** | 戦闘不能時に100%復活。復活時の魔獣数はスキルレベルで変動 | ガーネットベーゼ |
| **待機** | 発動ターンは攻撃せず、次ターン（or 2ターン後）の最初にスキルを先制発動 | ソウルスティール等 |

## 8. カード合成（出典: [スキル合成](https://seesaawiki.jp/kingdomconquest/d/%a5%b9%a5%ad%a5%eb%b9%e7%c0%ae)）

### 合成の基本

- **ベース(BASE)** カードに **素材(ADD)** カードを合成して強化
- 素材カードは **成否に関わらず消滅** する
- 合成には **DP（ダンジョンポイント）** または **CP（課金ポイント）** を消費

### スキル強化合成

**同じ魔獣（レアリティも同一）** を2枚合成して、スキルレベルを上げる。

- ベースが持つ全スキル（追加スキル含む）のいずれかを選んで強化可能
- スキルLvが高いほど成功率が下がる
- 最大 Lv10

### 追加スキルの習得

**別の魔獣** をADDとして合成し、新しいスキルを1つ習得する。

- ADDの魔獣の初期スキルに応じて、習得可能なスキル候補が3つ決まる
- 成功率（DP20使用時）: 35% / 10% / 5%
- 成功率（CP50使用時）: 70% / 20% / 10%
- ADDの初期スキルLvを2上げるごとに、候補がシフト（ローテーション）する
- スキルLv7以上で **上位スキル** が候補に追加される
- 同じスキルの重複習得は不可（確率が0%になり失敗率に加算）
- R/SRは初期スキルLvが3のため、候補のシフトが最初から進んでいる

### 追加スキルの削除

- ベースに任意のカードをADDとして合成し、追加スキルを1つ削除
- 初期スキル（Skill1）は削除不可
- 素材は何でもよいが、成否に関わらず消滅

### オーブ合成

- R向け: 同種族のR魔獣 + オーブで合成
- SR向け: 種族不問、SR + ハイオーブで合成
- 同名魔獣+オーブ両方消費で成功率+5%（Rの場合）or +10%（SRの場合）

### 強化魔獣（★）

- シーズン終了報酬やシャードから入手可能
- 射程以外の全ステータスが **10%増** 、生産コスト **25%OFF**
- 通常魔獣に同名同レアの★魔獣をADDすると、通常魔獣が★化する

## 9. 同盟システム（出典: [同盟](https://seesaawiki.jp/kingdomconquest/d/%c6%b1%cc%c1)）

### 同盟の基本

- ゲーム開始時、プレイヤー名の1人同盟に所属
- 他の同盟に加入するには、自分の同盟から全員脱退させた上で加入申請が必要
- 同盟員の領地は互いに攻撃不可
- `/ally` 宛てのメールで同盟員全員に送信可能

### 同盟レベル（出典: [同盟](https://seesaawiki.jp/kingdomconquest/d/%c6%b1%cc%c1)）

同盟員が資源を寄付してレベルアップ。ボーナスが増加する。

| 1 | 10人 | 0% | 0% | 10,000 |
| 2 | 15人 | 1% | 1% | 26,800 |

| 4 | 35人 | 2% | 2% | 118,800 |
| 5 | 45人 | 3.5% | 3.5% | 244,800 |

| 7 | 60人 | 4.5% | 4.5% | 984,800 |
| 8 | 65人 | 5% | 5% | 2,026,800 |
| 9 | 70人 | 5.5% | 5.5% | 4,182,800 |
| 10 | 75人 | 6% | 6% | 8,612,800 |
| 11 | 80人 | 6.5% | 6.5% | 17,626,800 |
| 12 | 85人 | 7% | 7% | 35,876,800 |

| 14 | 95人 | 8% | 8% | 146,611,800 |
| 15 | 100人 | 8.5% | 8.5% | 294,614,800 |

### 配下同盟システム

- 本拠地が他同盟に陥落すると、**配下同盟** にされる
- 自分が持っていた配下同盟は、攻撃した親同盟の配下に移る
- 配下同盟の状態で支配同盟の許可なく脱退すると、本拠地が消滅し最初からやり直し
- 配下同盟からの脱出には、第三者の協力による占領→編入→脱退の手順が必要

### 同盟戦（PvP）

- 同盟間で領地の奪い合いが発生
- 塔の占拠を巡って大規模な同盟戦が行われる
- 拠点には **耐久値** があり、勝利時の占拠力合計で削る
- 耐久値が 0 になると拠点が陥落（配下同盟化）

## 10. シーズン制（出典: [シーズン終了](https://seesaawiki.jp/kingdomconquest/d/%a5%b7%a1%bc%a5%ba%a5%f3%bd%aa%ce%bb)）

### シーズンの概要

- ゲームは **シーズン** 単位で進行（1st〜4th Season が存在した）
- シーズン終了は「World終了」と呼称。シーズン間にメンテナンス期間がある
- Season が進むにつれ、初期コスト値や伝承資料庫のコスト増加量が増えるなどの調整が入る

### シーズン終了時の処理

| 項目 | リセット | 引き継ぎ |
|---|---|---|
| 領地・拠点 | リセット | - |
| マップ配置 | リセット（ランダム再配置） | - |
| 魔獣カード | - | **引き継ぎ** |
| カードレベル・スキル | - | **引き継ぎ** |
| 装備品 | - | **引き継ぎ** |
| 同盟 | リセット | - |
| 資源 | リセット | - |
| CP（課金ポイント） | - | **返還処理**（出撃回数は1回100CPで払い戻し） |

### シーズン終了報酬

- シーズン終了時の成績に応じて **コンクエストパック** と **レジェンドパック** が配布される
- これらは次シーズン中に引くことができる
- 報酬から **強化魔獣（★）** を入手可能

## 11. 探索システム（出典: [探索](https://seesaawiki.jp/kingdomconquest/d/%c3%b5%ba%f7)）

### 概要

指定の土地に魔獣を派遣し、経験値・資源・DP・クリスタルなどを獲得する。

### 仕様

- 対象: 本拠地/拠点/塔以外の全ての土地（他プレイヤー占有地でも可）
- 所要時間: **約30分**（位置やステータスに関わらず固定）
- 探索後、魔獣のスタミナは **0** になる

### 探索結果

| 結果 | 説明 |
|---|---|
| **FAILURE** | 力尽き「休息中」状態に（3〜5時間行動不能） |
| **ALIVE** | 無事帰還。まれに1〜3時間ダウン |
| **SUCCEED** | 多めの報酬を獲得 |
| **EXCELLENT** | 大量の報酬 + ステータスポイント+1（名前末尾に「+」が付く） |

成功率は **魔獣のスタミナ** と **魔獣Lvと土地Lvの差** に影響される。

### 探索レベル

探索を繰り返してスコアを貯めるとレベルが上がり、同時派遣数が増加する。

| 探索Lv | 同時派遣数 |
|---|---|
| 1〜19 | 1体 |
| 20〜39 | 2体 |
| 40〜59 | 3体 |
| 60〜79 | 4体 |
| 80〜99 | 5体 |
| 100+ | 6体 |

### 特殊発見

- **ボス発見**: まれにボス魔獣を発見。RAID戦で戦闘可能
- **財宝発見**: まれに大量の資源・経験値・スコアを獲得
